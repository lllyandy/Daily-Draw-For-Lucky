# Contributing to Daily Draw: For Lucky 🎋

感谢您对 Daily Draw: For Lucky 项目的关注！
*Thank you for your interest in the Daily Draw: For Lucky project!*

## How to Contribute | 如何贡献

### Report Issues | 报告问题
If you find bugs or have suggestions, please submit an Issue:
如果您发现了 bug 或有改进建议，请提交 Issue：
1. Describe the issue | 描述问题
2. Steps to reproduce | 说明复现步骤
3. Environment info | 提供环境信息

### Submit Code | 提交代码
1. Fork this repository
2. Create your feature branch (`git checkout -b feature/AmazingFeature`)
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`)
4. Push to the branch (`git push origin feature/AmazingFeature`)
5. Open a Pull Request

### Improve Oracle Sticks | 签文改进
If you have better interpretations or story supplements for traditional oracle sticks, PRs are welcome.
如果您对传统签文有更好的解释或典故补充，欢迎提交 PR。

### Code Standards | 代码规范
- Use clear variable names
- Add necessary comments
- Keep code concise

## Code of Conduct | 行为准则

- Respect others | 尊重他人
- Accept constructive criticism | 接受建设性批评
- Focus on community best interests | 关注社区最佳利益
