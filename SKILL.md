# 观音灵签 - 每日抽签

🙏 **南无观世音菩萨**

每日一签，指点迷津。观音灵签共100签，涵盖人生各方面指引。

## 使用方法

### 触发方式
- 说：**"抽签"**
- 说：**"帮我抽个签"**
- 说：**"/draw"**

### 每日限制
- 每天只能抽取 **1次**
- 抽签结果会记录，次日可重新抽取

## 签文结构

每支签包含：
1. **签号** - 第X签
2. **吉凶** - 上上签/上签/中签/下签/下下签
3. **签诗** - 四句诗文
4. **解曰** - 整体解译
5. **详解** - 各方面指引
6. **典故** - 历史故事

## 100签吉凶分布

| 等级 | 数量 | 说明 |
|------|:----:|------|
| 上上签 | 约15支 | 大吉大利，万事如意 |

| 下签 | 约20支 | 凶多吉少，守成为上 |

## 解签指南

### 问事业
- 上上/上签：积极进取，把握时机
- 中签：稳扎稳打，不宜冒进
- 下/下下签：守成为上，静待时机

### 问姻缘
- 上上/上签：良缘天定，主动出击
- 中签：顺其自然，不可强求
- 下/下下签：缘分未到，修身养性

### 问健康
- 上上/上签：身体康健，注意保养
- 中签：小病无碍，劳逸结合
- 下/下下签：宜静养，遵医嘱

### 问财运
- 上上/上签：财源广进，可适当投资
- 中签：量入为出，谨慎理财
- 下/下下签：守财为主，不宜投机

## 示例签文

### 第一签 上上签 - 钟离成道

**签诗：**
```
开天辟地作良缘，吉日良时万物全；
若得此签非小可，人行中正帝王宣。
```

**解曰：** 大吉大利，万事亨通

**详解：**
- 事业：大展宏图，贵人相助
- 姻缘：天作之合，百年好合
- 健康：身强体健，百病不侵
- 财运：财源广进，富贵临门

**典故：** 钟离权，道教八仙之一，原为东汉大将，后得道成仙。
